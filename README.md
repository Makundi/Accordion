# Accordion
A lightweight, accessible accordion component built with vanilla JavaScript, HTML, and CSS. This project demonstrates DOM manipulation, event delegation, and smooth animations to create a clean, user-friendly interface where only one panel is expanded at a time.

Project Source: https://roadmap.sh/projects/accordion
